# usaskResearch
Houses project files for some test automation work for the university

Raspberry Pi files are to be run on the pi. These files contain:
* Socket Listener Script
* Arm compile jtag programming scripts
* Reset and helper Scripts

Verilog and fpga files are under the verilog folder. These files contain:
* Uart Core and address bus parsing
* Top level hookups to external pins and leds
* Pin definitions for externals in the hanyang board
